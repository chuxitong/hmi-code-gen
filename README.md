# Figma HMI Plugin

Это учебный проект в рамках ВКР. Задача — сделать плагин к Figma, который берёт макет промышленного HMI и с помощью визуально-языковой модели UI2Code^N превращает его в рабочий HTML/CSS. Плагин не пытается сделать идеальную конвертацию за один клик. Он должен помочь получить начальную версию кода, подтянуть её под референс итерациями и точечно править по текстовой команде вроде «сделай заголовок крупнее».

Репозиторий собран из трёх логических частей. В `figma-plugin/` лежит сам плагин: TypeScript-песочница `src/code.ts`, интерфейс `src/ui.html`, манифест и собранный `dist/code.js`. В `local-service/` работает HTTP-сервис на FastAPI с четырьмя эндпоинтами (`/generate`, `/refine`, `/edit`, `/render`) и двумя моделями: настоящей UI2Code^N (`model_wrapper.py`) и быстрым детерминированным заместителем (`rule_based_model.py`), чтобы пайплайн можно было запустить без GPU. В `baseline-tests/` хранятся скрипты экспериментов и их реальные выходы — HTML, PNG, JSON-метрики. Мокапы лежат в `mockups/`, еженедельные отчёты в `reports/`, текст ВКР в `thesis/`.

Чтобы поднять всё у себя, нужно сначала запустить локальный сервис: `cd local-service && .venv/Scripts/python.exe -m uvicorn app:app --port 8000`. Если есть желание использовать настоящую модель, нужно скачать её через `huggingface_hub` в `d:/hf_models/UI2Code_N` и выставить `USE_REAL_MODEL=1` перед запуском. После этого в Figma Desktop заходим в *Plugins → Development → Import plugin from manifest* и выбираем `figma-plugin/manifest.json`. Плагин откроется как панель, из неё можно генерировать и править код.

Тема ВКР на русском: «Разработка интеллектуального программного модуля генерации и итеративной корректировки кода человеко-машинных интерфейсов по графическим макетам». Полный текст и обзор — в `thesis/`.

Публичный репозиторий: https://github.com/chuxitong/hmi-code-gen

Воспроизводимость: из корня проекта `py baseline-tests/run_all_experiments.py`, затем проверка состава артефактов — `py baseline-tests/verify_deliverables.py`. Снимок прототипа с тегом `v2026.04.17-prototype-freeze` описан в `release-notes.md`.
