## Выполненные задачи

### 1. Изучение промышленных источников

Изучены следующие ресурсы для создания реалистичных промышленных макетов:

**Редактируемые шаблоны Figma:**
- Themesberg Figma Admin Dashboard Template
- Sneat Free Figma Admin Dashboard UI Kit

**Промышленные визуальные референсы:**
- Siemens HMI Template Suite — стандартные макеты операторских панелей
- Ignition SCADA Public Demo — реальные SCADA-дашборды
- FUXA Open-Source SCADA — синоптические экраны с датчиками и клапанами
- JointJS SCADA/HMI Demo — интерактивные диаграммы трубопроводов

### 2. Создание набора макетов (8 экранов)

| # | Название экрана | Промышленное применение | Сложность |
|---|----------------|------------------------|:---------:|
| 1 | Equipment Status Dashboard | Отображение состояния оборудования (работа, простой, неисправность) | Простой |
| 2 | Alarm & Event Screen | Список активных аварий с уровнями серьёзности и кнопками подтверждения | Простой |
| 3 | Real-Time Trend Monitor | Графики температуры, давления и расхода в реальном времени | Средний |
| 4 | Operator Control Panel | Панель ручного управления: кнопки пуск/стоп, уставки, селектор режимов | Средний |
| 5 | Production Line Overview | Обзор линии розлива: конвейеры, счётчики производительности, OEE | Средний |
| 6 | Tank Farm Synoptic | Синоптика резервуарного парка: уровни, клапаны, трубопроводы, насосы | Средний–Сложный |
| 7 | Energy Monitoring Dashboard | Мониторинг энергопотребления: круговые и столбчатые диаграммы, KPI | Средний–Сложный |
| 8 | Batch Recipe Management | Управление рецептом партии: последовательность шагов, таблица параметров | Сложный |

### 3. Дизайн-конвенции

Все макеты следуют стандартам промышленных HMI:
- Тёмный фон (#1E1E2E) для снижения нагрузки на глаза в операторских
- Цветовое кодирование: зелёный = норма, жёлтый = предупреждение, красный = авария, синий = информация
- Высокая контрастность текста на тёмном фоне
- Моноширинный шрифт для числовых значений
- Сетка отступов 8px / 16px, принципы ISA-101

## Текущие проблемы
